About CandidateOne

End-to-End Multi-Source Candidate Canonicalization Engine

Project Overview

CandidateOne is a modular ETL pipeline designed to consolidate candidate information collected from multiple recruitment platforms into a unified canonical profile.

The application extracts information from resumes, ATS exports, LinkedIn profiles, GitHub profiles and recruiter spreadsheets, normalizes inconsistent values, resolves conflicts, tracks provenance, calculates confidence scores and validates the final output before generating a canonical candidate profile.

Key Features

  • Resume PDF Extraction
  • ATS JSON Parsing
  • LinkedIn Profile Parsing
  • GitHub Profile Parsing
  • Recruiter CSV Parsing
  • Email & Phone Normalization
  • Location & Skill Standardization
  • Conflict Resolution
  • Confidence Scoring
  • Provenance Tracking
  • Configurable Projection Layer
  • JSON Schema Validation

Technology Stack

  • Python 3.13
  • Flask
  • Pandas
  • PyPDF2
  • Bootstrap 5
  • JSON Schema
  • HTML, CSS & JavaScript
  • Modular ETL Architecture
